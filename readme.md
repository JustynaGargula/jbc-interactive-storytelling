# JBC Interactive Storytelling

## Opis projektu

Projekt integruje metadane dokumentów życia społecznego JBC (Jagiellońska Biblioteka Cyfrowa) w graf wiedzy, tworzy powiązania semantyczne i wykorzystuje LLM do generowania narracji tematycznych, chronologicznych lub stylistycznych. Interfejs umożliwia eksplorację kolekcji w formie osi czasu, map lub ciągłych opowieści, wspierając kontekstowe odkrywanie treści w badaniach historycznych i edukacyjnych.

## Użyte technologie

- Python
- Streamlit
- Gemini API (google-genai)
