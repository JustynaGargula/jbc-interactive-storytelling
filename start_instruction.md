# Instalacja Gemini API

1. Wykonaj `pip install -q -U google-genai`
2. Utwórz klucz tutaj: [link](https://aistudio.google.com/app/apikey)
3. Zapisz ten klucz jako zmienną środowiskową: [instrukcja](https://ai.google.dev/gemini-api/docs/api-key?hl=pl#set-api-env-var)
4. Pamiętaj, żeby po tym uruchomić ponownie terminal i/lub IDE, zktórego korzystasz.
